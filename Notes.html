Intro to HTML

  • Write properly strcutured HTML docs
    • General rule
      • <tagname> Content </tagname>
    • Intro to MDN
      • Mozilla Developer Network
    • HTML Boilerplate and comments
      • Save as .html
      • document needs to have a boilerplate which can be achieve by typing <h & pressing the "Tab" key
      <!DOCTYPE html> - defines the document type as HTML
	<html> - HTML root element, everything needs to be inside the html element, 1 head followed by 1 body
		<head> - general info (metadata) about doc, including title and links to definitions of scripts
<!-- Comment --> - these are notes - 'Ctrl' + '/'
			<title>Webpage Title</title> - Name on Tab, SEO/links
		</head>
		<body> - where the content goes
<!-- Comment -->
		</body>
	</html>
  • Write common closing and self closing tags
    • Basic Tags
      • Heading Tags
        • Heading tags are block level elements and run from 1 to 6
        • <h1> Biggest heading tag</h1>
        • <h2> Secondary heading tag</h2>
        • <h6> Smallest heading tag</h6>
      • Paragraph Tags
        • Paragraph tags are block level elements used for displaying a paragraph
        • <p> This is a paragraph</p>
      • Adding bold or Italics
        • Bold or Italics should be added as a style, but can be used in HTML to emphasise something or making it strong
        • These tags are inline so they will not create a new line
        • <strong>Bold text goes here</strong>
        • <em>Italicised text goes here</em>
          • This line shows strong and emphasised text
    • Lists in HTML
      • There 2 types of lists that can be created in HTML5
        1. Ordered lists
          • Ordered lists are lists that are numbered, the format of the number can be defined within CSS
          • Ordered lists are the following format
            • <ol>
	<li>List item one</li>
	<li>List item two</li>
	<li>List item three</li>
</ol>
          • Here's how it would look
            1. List item one
            2. List item two
            3. List item three
        2. Unordered lists
          • Unordered lists are bullet pointed lists
          • Unordered lists are the following format
            • <ul>
	<li>List item one</li>
	<li>List item two</li>
	<li>List item three</li>
</ul>
          • Here's how it would look
            • List item one
            • List item two
            • List item three
          • Note the only the opening and closing tags change from <ol> to <ul>.
      • Lists can be nested within list just like this webpage
